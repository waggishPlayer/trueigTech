FitPlanHub Documentation

Welcome to FitPlanHub!

This project connects fitness trainers with users who want personalized workout plans. It is fully deployed and ready to use.

Application URL: https://fitplanhub.netlify.app

---

Project Overview

FitPlanHub allows users to browse and subscribe to fitness plans created by professional trainers. Trainers can sign up to create plans and manage their offerings, while users can find plans that fit their needs and follow their favorite trainers.

Features

For Users
- Browse a variety of fitness plans
- Subscribe to plans with a secure payment simulation
- Follow trainers to get updates in your personalized feed
- Advanced search to find exactly what you need

For Trainers
- Create and manage detailed workout plans
- Set pricing and duration for your plans
- Build a following of dedicated users

Technical Details

The application uses a modern stack to ensure performance and reliability:
- Backend: Node.js with Express and MongoDB for data storage.
- Frontend: Clean HTML, CSS, and JavaScript for a fast user experience.
- Deployment: Hosted on Netlify (Frontend) and Render (Backend).

Getting Started

To run this project locally, you will need Node.js installed. Clone the repository, install the dependencies using 'npm install', and start the server with 'npm run dev'.
